Cellular Automata Based Edge detector using Genetic Algorithm
Genetic Evolutionary Algorithms

Designed a GA which is capable of performing Edge detection on images by generating rule table based on edge detection algorithm required using python.

  • For genotype representation binary is used with population size of 50. Fitness evaluation is hamming distance between pixels is used. Parent selection is based on Tournament based and Survivor selection is based on (mu+lambda). Uniform crossover and uniform mutation is used for reproduction.
  • Achieved an accuracy of 95% while performing edge detection on images with respect to edge detection algorithm used.
Mar 2020 - May 2020

Puzzle solving using Genetic Algorithm
Genetic Evolutionary Algorithms

Given a rectangular frame of size L(length) and W(width) and puzzle pieces (N), developed GA algorithm arranges the pieces in such a way that it minimizes the free space within the frame using python.

  • For genotype representation integer is used with population size of 300 individuals. Fitness function is based on area minimizing. Parent selection is based on exponential ranking and survivor selection is based on mu+lambda strategy. Uniform cross-over and non uniform mutation is used for reproduction.
  • The resulting fitness is 94 percent for complex puzzles with small pieces to fit.
